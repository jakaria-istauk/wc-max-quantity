== Description ==

The WC Max Quantity plugin allows you to set a maximum quantity limit for each product in your WooCommerce store. With this plugin, you can control the maximum number of units that can be added to the cart for each product, and prevent customers from adding more than the specified limit.

The plugin is easy to use and configure. Simply go to the "Products" page in your WooCommerce store, select the product you want to set a maximum quantity limit for, and enter the limit in the "Maximum Quantity" field.

== Installation ==

    01. Download the plugin zip file from the WordPress plugin repository.
    02. Log in to your WordPress website and go to the "Plugins" page.
    03. Click on the "Add New" button at the top of the page.
    04. Click on the "Upload Plugin" button and select the zip file you downloaded in step 1.
    05. Click on the "Install Now" button and wait for the installation to complete.
    06. Activate the plugin by clicking on the "Activate" button.

== Frequently Asked Questions ==

= How do I set a maximum quantity limit for a product? =

To set a maximum quantity limit for a product, go to the "Products" page in your WooCommerce store, select the product you want to set a limit for, and enter the limit in the "Maximum Quantity" field under the "Inventory" tab.
